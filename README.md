# Responsive Landing Page

A clean, modern, and fully responsive landing page built with HTML and CSS. This project demonstrates responsive design principles, including Flexbox and Grid layouts, media queries for mobile adaptation, and CSS-only navigation toggle.

## Features

- **Responsive Design**: Adapts seamlessly to desktop, tablet, and mobile screens using media queries.
- **CSS-Only Navigation**: Hamburger menu for mobile without JavaScript.
- **Modern UI**: Gradient backgrounds, smooth animations, and hover effects.
- **Accessibility**: Semantic HTML, ARIA labels, and keyboard-friendly navigation.
- **No JavaScript Required**: Pure HTML & CSS implementation for core functionality.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties, Animations)

## How to Run Locally

1. Clone the repository:
   ```
   git clone https://github.com/Karthikn-code/Task-1.git
   cd landing_page
   ```

2. Open `index.html` in your browser, or use a local server:
   - Install live-server: `npm install -g live-server`
   - Run: `live-server`
   - Open http://127.0.0.1:8080 in your browser.

## Testing Responsiveness

- Resize your browser window to test breakpoints.
- Use browser developer tools (F12) > Toggle device toolbar to simulate mobile devices.
- Recommended Chrome extension: "Responsive Viewer" for easy device simulation.
